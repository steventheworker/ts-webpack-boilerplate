# typescript + webpack minimal config
```shell
npm install
```
### installs as devDependencies in package.json:
####	webpack
		- webpack-dev-server
		- webpack-cli
		- html-webpack-plugin
####	typescript
		- ts-loader
		- @types/html-webpack-plugin

start webpack-dev-server
```shell
npm run serve
```

build files to upload to static web server
```shell
npm run build
```
